# qq小程序

### 获取地理位置

```js
 qq.getLocation({
        type: 'wgs84',
        success(res) {
          const latitude = res.latitude
          const longitude = res.longitude
          const speed = res.speed
          const accuracy = res.accuracy
        }
      })
```

需要在app.json中添加permission字段

app.json

```json
"permission":{
    "scope.userLocation":{
      "desc":"打开位置共享"
    }
  }
```
